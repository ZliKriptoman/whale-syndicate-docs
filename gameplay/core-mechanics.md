## üìä Level System

### How Leveling Works

**Current Level** determines:
1. **Tap Power** (1 coin = 1 level)
2. **Passive Income Cap** (PPH multiplier)
3. **Upgrade Availability** (higher tiers unlock)
4. **Offline Earning Limits** (storage caps)

### Level Cost Formula

**Price to upgrade from current level:**
```
Cost = 500 √ó (1.22 ^ Current Level)
```

**Examples:**

| Current Level | Upgrade Cost | Cumulative Total |
|---------------|--------------|------------------|
| 1 ‚Üí 2 | 610 | 610 |
| 10 ‚Üí 11 | 4,330 | ~30K |
| 25 ‚Üí 26 | 58,600 | ~1M |
| 50 ‚Üí 51 | 1.47M | ~50M |
| 75 ‚Üí 76 | 36.9M | ~1.5B |
| 99 ‚Üí 100 | 1.18B | ~50B |

**Translation:** Each level costs **22% more** than the last.

### Exponential Scaling

**Why 22%?**

- Prevents instant Level 100
- Rewards daily consistency
- Makes premium packages valuable
- Ensures 3-4 month grind (with Happy Fifth)

**Math Fact:** To reach Level 100 from scratch requires approximately **50 billion coins** total.

## ü§ë Passive Income (PPH)

### What is PPH?

**PPH** = **Profit Per Hour**

**Passive earnings while offline OR online.**

**Display:**
```
üìà PPH: 125,000/hr
```

### How PPH Works

**Every hour you accumulate:**
```
Balance += PPH
```

**Even when:**
- App is closed
- Phone is off
- You're sleeping
- You're on holiday

**Automatic wealth generation!**

### PPH Sources

**1. Upgrades:**
- Buy passive income equipment
- Each upgrade adds to PPH
- Class-dependent items

**2. Referrals:**
- +10% PPH per 10 referrals
- Stacks infinitely
- 100 referrals = +100% PPH (DOUBLE!)

**3. Level Bonuses:**
- Higher level = better upgrades
- Unlocks legendary equipment

### PPH Calculation

**Base PPH:**
```
Sum of all passive income upgrades
```

**With Referral Bonus:**
```
Total PPH = Base PPH √ó (1 + Referral_Bonus)
```

**Example:**
```
Base PPH: 100,000/hr
Referrals: 50 (= +50% bonus)
Total PPH: 100,000 √ó 1.5 = 150,000/hr
```

{% hint style="success" %}
**Referrals are CRUCIAL for PPH scaling.**

10 refs = +10%
50 refs = +50% 
100 refs = +100% (DOUBLE your passive income!)

## üé∞ Degen Flip

### Overview

**50/50 coin flip gambling**

**Bet your coins. Double or nothing.**

### Rules

**Cost:** Any amount (minimum ~1,000)

**Outcome:**
- üü¢ **Win:** Get 2x your bet
- üî¥ **Lose:** Lose entire bet

**Daily Limit:** 50 flips/day

**Refill Options:**
- Watch ad (+5 flips)
- Pay 1 TON / 100 Stars (+10 flips)

### Math

**House Edge:** ~2% (fair game!)

**Expected Value:**
```
EV = 0.49 √ó (2x) + 0.51 √ó (0x) = 0.98x
```

**Translation:** On average, you lose 2% per flip.

**Over 1,000 flips:**
```
Expected loss = 2% of total volume
```

### Strategy

{% tabs %}
{% tab title="Casual Players" %}

**Use sparingly.**

- Max 10 flips/day
- Bet small amounts (1-5% bankroll)
- Stop after 3 losses in a row

**Why:** Variance kills bankrolls.

{% endtab %}

{% tab title="Grinders" %}

**Avoid entirely.**

- Tapping is more +EV
- Happy Fifth = guaranteed gains
- Flipping = -EV long-term

**Exception:** Daily mission if required.

{% endtab %}

{% tab title="Degens" %}

**GO ALL IN.**

- YOLO entire bankroll
